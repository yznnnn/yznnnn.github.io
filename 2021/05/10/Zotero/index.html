    1. 介绍

    Zotero是一款自由及开放源代码的文献管理软件，管理书目信息（如作者、标题、出版社、摘要、阅读笔记等）及相关材料（如PDF文件等)。其最著名的特性是作为浏览器插件、在线同步、与文档编辑软件如Microsoft Word、LibreOffice、OpenOffice.org Writer、NeoOffice等集成，可生成文内引用、生成页面脚注或文后的参考文献（bibliographies）。

    “Zotero”大致来源于阿尔巴尼亚语动词，意为“掌握”（to master）

    Zotero 支持全平台，完全免费，个人认为是最好的文献管理软件。

    2. 下载安装

    在这个页面下载你对应系统的Zotera进行安装，本文是在 Win10 环境下安装的。

    3. 初始设置

    3.1. 文件保存目录设置

    点击顶部菜单栏，编辑(E) > 首选项(N) > 高级 > 文件和文件夹 > 数据存储位置 处设置。

    在你更换计算机或重装系统时，可以直接备份这里的数据文件，复制到新的设备或系统中即可继续使用。

    4. 插件安装

    4.1. ZotFile

    ZotFile 是一款用于自动重命名PDF的插件。

    在官网下载 ''zotfile-5.0.16-fx.xpi'' 后，打开 Zotero，点击顶部菜单栏，工具(T) > 插件 > ⚙ > install add-on from file ，选择刚刚下载的xpi文件。

    安装完重启软件，就能在 工具(T) 里看到插件设置选项 ZotFile Preference。

    打开后选择 Renaming Rules ，就能看见文件命名规则。

    我的命名规则为：

    {%y_}{%a_}{%t}
    • %y：年份
    • %a：作者
    • %t：标题
