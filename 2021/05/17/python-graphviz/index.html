    介绍

    Graphviz （英文：Graph Visualization Software的缩写）是一个由AT&T实验室启动的开源工具包，用于绘制DOT语言脚本描述的图形。它也提供了供其它软件使用的库。Graphviz是一个自由软件，其授权为Eclipse Public License。其Mac版本曾经获得2004年的苹果设计奖[2]。

    安装

    conda install pydot
conda install graphviz
conda install python-graphviz

    在 Win10 中，还需要在环境变量 path 中添加类似 C:\Users\yzn\anaconda3\Library\bin\graphviz 的内容。

    使用文档

    官方使用文档

    一个简单的用在 Jupyter Lab 中的例子

    import graphviz
d = graphviz.Digraph()
d.node("node_1")
d.node("node_2")
d.edge("node_1","node_2")
d
