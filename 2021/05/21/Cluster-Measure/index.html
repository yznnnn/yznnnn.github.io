    外部评价指标

    聚类算法中的外部评价指标指的是，在评价聚类结果时，还需要真实的数据标签。

    给定数据集 \({\mathbf X}\in{\mathbb R}^{m\times n}\)，\(n\) 个列向量表示 \(n\) 个样本，每个样本有 \(m\) 维特征，对应的真实标签为 \({\mathbf Y}\in{\mathbb R}^{n\times 1}\)。

    经过某个聚类算法得到的预测标签为 \(\overline{\mathbf Y}\in{\mathbb R}^{n\times 1}\)

    Purity

    \[ {\rm Purity}(\Omega,C) = \frac{1}{n} \sum_k\max_z|\omega_k\cap c_z| \in[0,1] \]

    其中：

    • \(\Omega\) 是基于 \(\overline{\mathbf Y}\) 对样本按预测得到的簇进行的预测簇划分： \[ \Omega=\{\omega_1,\omega_2,...,\omega_K\}\\ \omega_k=\{ {\mathbf x}_i |\overline{y}_i=k \} \]

    • \(C\) 是基于 \({\mathbf Y}\) 对样本按真实分类进行的真实簇划分： \[ C=\{c_1,c_2,...,c_Z\}\\ c_z=\{ {\mathbf x}_i |y_i=z \} \]

    • \(\max_z|\omega_k\cap c_z|\) 的意思是，在集合 \(\omega_k\) 中，对于每个真实标签 \(z\) 而言，计算满足 \(y_i=z\) 的 \({\mathbf x}_i\) 的数量，取最大值： \[ \omega_k\cap c_z= \{ {\mathbf x}_i| \overline{y}_i=k {\ \rm and\ } y_i=z \} \]

    function purity = Purity(Label,Label_predict)
%UNTITLED 此处显示有关此函数的摘要
%   此处显示详细说明
Lidx = unique(Label); 
predLidx = unique(Label_predict); 
pred_classnum = length(predLidx);
correnum = 0;


for ci = 1:pred_classnum
    incluster = Label(find(Label_predict == predLidx(ci)));
    inclunub = hist(incluster, 1:max(incluster)); 
    if isempty(inclunub) 
        inclunub=0;
    end
    correnum = correnum + max(inclunub);
end
purity = correnum/length(Label_predict);
end
