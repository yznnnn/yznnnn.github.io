    1. 介绍

    关于 Anaconda ，维基百科是这么介绍的。

    Anaconda 是一个免费开源的Python和R语言的发行版本，用于计算科学（数据科学、机器学习、大数据处理和预测分析），Anaconda致力于简化软件包管理系统和部署。Anaconda的包使用软件包管理系统Conda进行管理。超过1200万人使用Anaconda发行版本，并且Anaconda拥有超过1400个适用于Windows、Linux和MacOS的数据科学软件包。

    简而言之，我的理解就是，对于从事数据科学相关行业并需要使用Python语言的人而言，Anaconda提供了开箱即用的便捷，集成了大部分数据处理包，免除了单独下载Python后需要一个个安装自己的包的麻烦。其次，Anaconda提供了方便的环境管理，通过虚拟环境的方式，使你能够基于不同项目构建互不冲突的个性化Python环境（Python与工具包的版本不同）。

    自2019年接触机器学习以来，Anaconda便是我电脑中必须要装的软件，重要性等同于QQ！

    下面将介绍我是如何在Win10上搭建Anaconda环境的，仅供参考，毕竟每个人使用习惯不一样。

    2. 安装

    在 Anaconda 的官网可以直接下载最新版本的 Anaconda，如果你觉得太慢，也可以下载清华镜像的版本。

    在 Win10 中，下载后直接点击安装就好。（注意，安装时不要选择将地址加入环境变量）

    3. 初始配置

    3.1. Anaconda Prompt 设置（非必须）

    Anaconda Prompt (anaconda3) 是 Anaconda3 提供的默认命令行，你可以直接使用。

    如果你在 Win10 中安装了 Windows Terminal，那么接下来将介绍如何在 Windows Terminal 中使用 Anaconda Prompt (anaconda3)。该步可以跳过，不影响后续步骤。

    （关于 Windows Terminal 的安装，直接在微软商店搜索就好）

    在桌面右键鼠标，打开 Windows Terminal，点击标题栏的 + 旁边的向下的三角，选择设置，在打开的界面左下有一个 ⚙ 打开JSON文件 ，点击后可以打开 Windows Terminal 的配置文件。（推荐使用 Vs Code 进行编辑）

    仿照下面的设置就可以了：

    "profiles":
{
    "defaults":
    {
        // Put settings here that you want to apply to all profiles.
    },
    "list":
    [
        {
            // Make changes here to the cmd.exe profile.
            "guid": "{0caa0dad-35be-5f56-a8ff-afceeeaa6101}",
            "name": "命令提示符",
            "commandline": "cmd.exe",
            "hidden": false
        },
        {   // Anaconda Settings
            "guid": "{b453ae62-4e3d-5e58-b989-0a998ec441b7}",
            "hidden": false,
            "name": "Anaconda Prompt",
            "icon": "C:\\Users\\xxx\\anaconda3\\Menu\\anaconda-navigator.ico",
            "commandline": "cmd.exe /K C:\\Users\\xxx\\anaconda3\\Scripts\\activate.bat"
        },
        {
            "guid": "{b453ae62-4e3d-5e58-b989-0a998ec441b8}",
            "hidden": false,
            "name": "Azure Cloud Shell",
            "source": "Windows.Terminal.Azure"
        }
    ]
},

    3.2. 查看 Anaconda 版本

    打开 Anaconda Prompt (anaconda3) ，执行如下指令即可查看当前安装的 Anaconda 的版本。

    conda --version

    我的运行结果为

    (base) C:>conda --version conda 4.9.2

    3.3. 设置国内源（非必须）

    由于墙的存在，需要联网的指令操作会慢的令人抓狂，这里给出借用 清华镜像 与 中科大镜像 的设置方法。

    查看已经安装的镜像源：

    conda config --show

    添加清华源：

    # 添加清华源
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
# pytorch
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud/pytorch/
# 设置搜索时显示通道地址
conda config --set show_channel_urls yes

    添加中科大源：

    # 添加中科大源
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/pkgs/main/
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/cloud/conda-forge/
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/cloud/msys2/
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/cloud/bioconda/
conda config --add channels https://mirrors.ustc.edu.cn/anaconda/cloud/menpo/
conda config --set show_channel_urls yes

    恢复默认源：

    conda config --remove-key channels

    3.4. 更新 Anaconda 至最新

    conda update conda
conda update anaconda

    4. 安装一些必要的包

    4.1. graphviz

    这是一个用于绘图的包。

    conda install pydot
conda install graphviz
conda install python-graphviz

    在 Win10 中，还需要在环境变量 path 中添加类似 C:\Users\yzn\anaconda3\Library\bin\graphviz 的内容。
